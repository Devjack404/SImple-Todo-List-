# TODO LIST 
## PROYEK TODO LIST MENGGUNAKAN VUE.JS DAN NUXT

## Apa ini ?
Ini adalah proyek website saya menggunakan Vue.js dan Nuxt.js. Saya mencoba membuat TODO LIST yang memiliki desain yang sanga simple menggunakan Tailwind
saya harap proyek ini akan terus berkembang terus :>

## Cara Menjalankan ?
1. Bisa menggunakan code berikut :
   ```sh
   git clone https://github.com/Devjack404/SImple-Todo-List-.git
   ```
2. Masuk Folder :
   ```sh
   cd SImple-Todo-List-
   ```
3. Buka Code Editor (VS CODE)
   ```sh
   code .
   ```
4. Jalankan Websitenya dan masuk ke
localhost yang tersedia yaitu di port (3000) :
   ```sh
   npm run dev
   ```
